Projektemben egy zombis lövöldözős játékot készítettem, amit
a w,a,s,d billentyűkkel és az egérrel irányítva játszhatunk
A program összetett logikát használ: a zombik véletlenszerűen jelennek meg a pálya szélein
és folyamatosan a karakter felé tartanak, miközben a lövedékek pontosan követik a
játékos nézési irányát. A túlélést egy 5 pontos életerő-rendszer és sérülés utáni
rövid láthatatlanság segíti, a pontszám pedig minden sikeres kiiktatás után növekszik,
egészen a "Game Over" képernyő megjelenéséig.
